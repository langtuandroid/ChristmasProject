Detailed Description

AgentCharacter.


The AgentCharacter is a customized Character derived class adding navigation and path finding capabilities (NavMesh, NavMeshAgent). This allows you to create characters that can intelligently move around the game world, using navigation meshes (player or AI controlled). By default, this implements a typical click-to-move behavior.
