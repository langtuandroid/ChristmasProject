Detailed Description

Overrides a CharacterMovement SlopeLimit property allowing to define per-object behavior instead of per face.

This enable you to tweak what surfaces Characters can walk up. Perhaps a stair case is too steep or maybe you want to enforce the "no walking on the grass" signs, these settings will enable you to do so.
