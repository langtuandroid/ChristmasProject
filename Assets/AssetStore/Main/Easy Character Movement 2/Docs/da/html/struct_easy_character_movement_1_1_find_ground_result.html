◆ surfaceNormal

Vector3 EasyCharacterMovement.FindGroundResult.surfaceNormal

Normal of the hit in world space, for the object that was hit by the sweep, if any.

For example if a capsule hits a flat plane, this is a normalized vector pointing out from the plane. In the case of impact with a corner or edge of a surface, usually the "most opposing" normal (opposed to the query direction) is chosen.
